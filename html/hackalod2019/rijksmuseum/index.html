Data Stories over met collectiegegevens van het Rijksmuseum

Rijksmuseum collectie

Dit is een experimentele dataset opgebouwd uit de gegevens die beschikbaar zijn via de OAI-PMH Api van het Rijksmuseum. De gegevens zijn afkomstig uit de export in het formaat 'edm_dc'. De gegevens bevat een flink deel van de beschrijving uit de set 'PublicDomainImages', deze bevat alle vrij beschikbare objecten waarvan een ook een afbeelding beschikbaar is.

De data is tijdelijk beschikbaar gemaakt in het NDE LOD Platform. Momenteel bevat de data op het NDE LOD Platform ca. 250.000 objectbeschrijven, dit is echter niet de volledige collectie informatie, zie voor alle informatie de Rijkstudio.

De volgende query laat zien hoe schilderijen gevonden kunnen worden via de naam van de maker en op hoe de bijbehorende details en afbeelding te vinden zijn.
