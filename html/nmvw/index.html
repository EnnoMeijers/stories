Overzicht van de LOD dataset van het Nationaal Museum voor Wereldculturen

Het Nationaal Museum voor Wereldculturen heeft op zijn website zijn collectie beschikbaar gesteld in LOD-formaat. De dataset bevat de metadata over de objecten in de collectie en de termen in de NMvW-thesaurus. De data is gedownload en opgenomen in de data-omgeving van NDE zodat de data kan worden bevraagd via een SPARQL-endpoint. Hier is ook in detail uitgelegd wat de dataset is en waar deze te verkrijgen is. In deze datastory leggen we uit wat je in de dataset kunt verwachten en wat voor vragen je zou kunnen stellen.
